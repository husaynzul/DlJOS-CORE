# ➕ PLATFORM BUTTON LOGIC

When user clicks "+ Connect":

1. Open API/OAuth connection flow
2. Ask for API key OR login permission
3. Validate connection
4. Set status = "connected"
5. Update UI instantly

When user clicks "Disconnect":

1. Remove token securely
2. Set status = "not_connected"
3. Keep platform visible
4. Restore "+ Connect" button

IMPORTANT:
Buttons must ALWAYS work. No dead UI elements allowed.
